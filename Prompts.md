# Prompts Used (Learning Style)

1. I asked: *"Why don’t I see any CSS styling on my page? Can you explain what might be missing and how to fix it?"*

2. I asked: *"How do I correctly add a logo to my navbar so it shows up on the left side?"*

3. I asked: *"Does the code you gave me include all the Level 1 requirements like responsiveness, navbar, hero section, services, and footer?"*

4. I asked: *"Can you explain step by step how to push my project to GitHub after creating a repo?"*

5. I asked: *"Why am I facing errors when I try to push my code — what do they mean and how do I fix them?"*

6. I asked: *"How should I structure my README.md so reviewers can understand my project clearly?"*

7. I asked: *"Can you help me phrase my prompts in a more natural, learning‑focused way instead of just asking for code?"*
